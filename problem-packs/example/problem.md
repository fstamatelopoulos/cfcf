# Example Problem: Build a Calculator Module

Build a simple calculator module in TypeScript that supports basic arithmetic operations.

## Requirements

- Create a `calculator.ts` module with functions: `add`, `subtract`, `multiply`, `divide`
- Each function takes two numbers and returns a number
- Division by zero should throw an error with a clear message
- Export all functions from the module
- The module should be importable as `import { add, subtract, multiply, divide } from "./calculator"`

## Existing Context

This is a blank project. There is no existing code.
