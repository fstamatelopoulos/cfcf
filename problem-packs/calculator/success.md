# Success Criteria

## Tests Must Pass

- `add(2, 3)` returns `5`
- `subtract(10, 4)` returns `6`
- `multiply(3, 7)` returns `21`
- `divide(15, 3)` returns `5`
- `divide(10, 0)` throws an error
- All functions handle negative numbers correctly
- All functions handle decimal numbers correctly

## Code Quality

- TypeScript with proper type annotations
- Functions are exported and importable
- Error messages are descriptive
